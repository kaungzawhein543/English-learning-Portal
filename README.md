## Features

- Add and manage custom English learning topics
- View topics in a hierarchical tree structure
- Add vocabulary words with definitions for each topic
- Add grammar notes categorized by CEFR levels (A1-C2) with detailed patterns, explanations, and example sentences
- Search and filter functionality for both Topics and Grammar Notes
- Modern, professional UI with smooth animations, responsive design, and a Dark Mode toggle
- Data persistence using local storage

## Getting Started

Simply open `index.html` in your web browser. No server required.

## Usage

- Switch between "Topics" and "Grammar Notes" tabs
- Toggle between Light and Dark mode using the sun/moon icon in the header
- Search for specific topics or grammar notes using the search bars
- Add new topics with titles, descriptions, content, and vocabulary
- Organize topics hierarchically by selecting parent topics
- Click "Add Vocab" on any existing topic to add more vocabulary words
- Add comprehensive grammar notes (including pattern, explanation, and examples) for different proficiency levels
- All data is saved locally in your browser
